Amae

  • Ideation,UI/UX Design
  • Tools: Photoshop, Figma
  • date: Summer 2020
  • Group Project

Amae is a utility app for busy parents. Amae helps parents to manage their time, learn about parenting and get help easily and quickly.

Ideation

It all began with selecting a service of exchanging economies. We then defined our design problem statement and took it towards various participant groups. We collected primary and secondary data from which we made some discoveries. Based on them we took it to work on some prototypes. Some of the testing methods have been stated below.

We really wanted to provide an experience which would allow the user to multitask and deligate work which would help in reducing time. The allowing the user to spend more time with their child. The project was super fun to work on and got real big.

Reflections

This was a group project which helped me to understand UX/UI design and create new prototypes. Moreover, the use of Figma alos taught me a lot of things which our team collaborated through that will allow me to use these skills in further projects.
